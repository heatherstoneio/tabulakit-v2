# Welcome

Welcome to your new documentation site, powered by [TabulaKit](https://github.com/heatherstoneio/tabulakit).

## Getting Started

If you haven't already, run the `/startup` wizard in Claude Code to configure your site:

```
/startup
```

The wizard will help you set your site name, choose a theme, select a deployment target, and pick a content template.

## Manual Setup

You can also configure your site manually by editing:

- **`config.js`** — Site name, description, theme colors
- **`auth-config.js`** — Authentication settings (public, domain-restricted, or email allowlist)
- **`_sidebar.md`** — Navigation structure

## Local Preview

```bash
npx live-server site --port=3000
```
