## About This Scaffold

This is the OPORD 202604-019 Phase 1 scaffold — the minimum viable Astro/Starlight site on top of the v2 development fork. It demonstrates:

- **Content location:** `src/content/docs/`
- **Sidebar:** configured in `astro.config.mjs`
- **Dark/light toggle:** built-in via Starlight
- **Site title:** single config value (`TABULAKIT_TITLE` env var or `SITE_TITLE` constant in `astro.config.mjs`)
- **Node version:** pinned via `.nvmrc` (22 LTS)
- **Deploy:** GitHub Actions → GitHub Pages (per OPORD 019 Phase 2)

Deeper template features (themed presets, Setup tab integration, tree-nav panels) ship in [OPORDs 020-022](https://gitlab.com/heatherstone/mojo/-/issues/?label_name%5B%5D=plan%3A%3A202604-004).
